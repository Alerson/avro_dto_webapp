Avro to DTO Generator

Esta aplicação web converte esquemas Avro (.avsc) em classes Java DTO (Data Transfer Object).

Sobre esta aplicação

O Avro to DTO Generator permite converter facilmente seus esquemas Avro em classes Java DTO adequadas para uso em seus projetos.

Como usar:

  1. Faça upload de um arquivo de esquema Avro (.avsc)
  2. Clique em "Gerar DTO"
  3. O arquivo Java será gerado e baixado automaticamente para seu computador

Recursos:

  • Converte tipos Avro em tipos Java apropriados
  • Suporta registros aninhados e enumerações
  • Gera getters e setters
  • Gera construtores

Upload de Esquema Avro

O arquivo deve ser um esquema Avro válido no formato JSON.
